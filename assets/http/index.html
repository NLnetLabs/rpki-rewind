RPKI Wayback Machine 2.0

The main idea of this database is that you can request the state of the RPKI at any random point in time and recreate it locally in a form that can be parsed by a relying party client (e.g. Routinator). Data is fetched at a 1-minute interval.

Generating the tar file might take a couple of minutes. The page may seem unresponsive but things are actually happening.
